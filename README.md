# Personal Developer Portfolio

A clean, modern, and responsive personal portfolio website built with HTML, CSS, and JavaScript.

## üöÄ Quick Start

### Prerequisites
- A code editor (VS Code recommended)
- A modern web browser

### Local Development
Since this project uses vanilla HTML/CSS/JS, you can simply open the `index.html` file in your browser.

For a better experience (auto-reload), use a simple HTTP server.
If you have Python installed:
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

If you have Node.js installed, you can use `http-server` or the VS Code Live Server extension.

## üì¶ Deployment (GitHub Pages)

This project is ready to be hosted on GitHub Pages for free!

1. **Create a Repository**: Create a new public repository on GitHub (e.g., `my-portfolio`).
2. **Push Code**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to your repository **Settings**.
   - Navigate to the **Pages** section (on the left sidebar).
   - Under **Source**, select `main` branch.
   - Click **Save**.

Your site will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/` shortly!
